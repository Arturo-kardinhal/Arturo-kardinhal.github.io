4.6 Direcciones IP y nombres de Dominio

Cada dispositivo en Internet tiene una dirección IP, que es como su “dirección postal” digital. Existen dos tipos principales:

  • IPv4: Direcciones de 32 bits (ejemplo: 192.168.1.1). Es la versión tradicional, aunque su espacio se está agotando.
  • IPv6: Direcciones de 128 bits, ofreciendo prácticamente infinitas combinaciones (ejemplo: 2001:0db8:85a3:0000:0000:8a2e:0370:7334).

Para no tener que recordar largos números, usamos nombres de dominio (como www.google.com). El sistema DNS traduce esos nombres en direcciones IP, facilitando el acceso a los sitios web.
